# Cpp-project
INTRODUCTION
This project is a Railway Route Management System implemented using C++.
It helps to manage railway stations and their details like distance and fare using a linked list data structure.

The system allows users to:
Add and remove stations
View and search routes
Calculate total distance and fare
Simulate travel and maintain history

The project demonstrates the practical use of:
Linked Lists
Stack
Dynamic memory allocation

OBJECTIVES
To understand linked list operations (insertion, deletion, traversal)
To implement stack operations (push and pop)
To develop a menu-driven program
To simulate real-world railway route management
To improve problem-solving and programming skills
